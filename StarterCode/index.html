Correlations Discovered Between Health Risks and Income

This charts depicts information from the U.S. Census Bureau and the Behavioral Risk Factor Surveillance System.The data set used is based on 2014 ACS 1-year estimates: https://factfinder.census.gov/faces/nav/jsf/pages/searchresults.xhtml

The purpose is to find health risks facing particular demographics. The current data set includes data on rates of percentage of population in Poverty and percentage of population that lacks Healthcare. Using a statescatter plot we visualize correlation between Healthcare vs. Poverty

There is a slight positivie correlation: the higher the percentage of population in poverty, the higher percentage population that lacks healthcare.
